# 3.6 Batch (英文版 / English Version)

Batch program are all derived from base class BatchProgram, or CalledProgram.

A batch program is used to handle flat files and sorting. They do not have access to CICS like environment.

To achieve that purpose, the following declarations are available:

	- File definition

	- File verbs handling

	- Sort support

As of 03/2007, only sequential data access is supported by nacaRT.
